# TimeStamp Method
Sets an DateTime value (e.g. CurrentDateTime()) as the timestamp of a ExceptionResponse object.

## Syntax
```javascript
	TimeStamp(dt: DateTime)
```

## Parameters
### *dt*
Type: DateTime<br/>
Timestamp as DateTime to be assigned to the 'timestamp' exception property.

## Return Value
### *None*

# ConcatenateText Method
Concatenate text with whitespace in between.

## Syntax
```javascript
	[Text := ] ConcatenateText(textPart1: Text; textPart2: Text)
```

## Parameters
### *textPart1*
Type: Text<br/>
First part of a text as string.
### *textPart2*
Type: Text<br/>
Second part of a text as string.

## Return Value
### *Text*
Concatenated text.

# GetApiVersion Method
Returns the currently used API Version.

## Syntax
```javascript
    [Text := ] GetApiVersion()
```

## Parameters
### *None*

## Return Value
### *Text*
Type: Text<br/>
The version of the 365 business API metadata.

## Remarks
Returned format is `MAJOR`.`MINOR`.`BUILD`.`REVISION`.

# Decode Method
Decodes the barcode.

## Syntax
```javascript
	Decode(var tempBlobBuffer: Codeunit "Temp Blob"; var barcodeValue: Text)
```

## Parameters
### *tempBlobBuffer*
Type: Codeunit "Temp Blob"<br/>
Temp Blob codeunit instance to storing the barcode to encode.
### *barcodeValue*
Type: Text<br/>
Out variable for encoded Barcode Value.

## Return Value
### *None*
